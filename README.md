# VR Catalog

Современный сервис просмотра товарного каталога с импортом из XML в базу данных.

## Архитектура

- `backend/` — FastAPI REST API, SQLAlchemy-модели, SQLite и Alembic.
- `backend/app/importer/` — независимый модуль импорта XML. Только он читает XML; поиск, фильтры и экспорт работают по базе данных.
- `frontend/` — React + TypeScript + Vite + Material UI.

## Быстрый запуск

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

```bash
cd frontend
npm install
npm run dev
```

## Возможности

- Ручная загрузка XML с полной заменой предыдущего каталога.
- Импорт товаров, цен любых типов, остатков по складам, всех свойств, изображений, аналогов и штрихкодов.
- Поиск без учета регистра по названию, коду, артикулу, описанию, бренду, производителю, штрихкодам и тегам.
- Автоматически формируемые фильтры по данным базы.
- Карточки товаров, детальная карточка, избранное, история просмотров, копирование полей.
- Экспорт найденных товаров в CSV и Excel.
- Хранение последних 10 импортов как база для будущего отката и сравнения версий.
